# Silverscreen

## Getting Started

### Install .Net Core
Install Dotnet Core

Download project dependencies by running `dotnet restore`.

### Install NodeJS
Seriously, just install NodeJS & NPM

### Install Aurelia-CLI
`npm install -g aurelia-cli`

### Build UI
`cd ui`
`npm install`
`au build`

### Run Project
`cd ..`
`dotnet run`
